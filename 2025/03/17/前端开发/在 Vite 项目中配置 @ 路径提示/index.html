常见问题排查

  1. Vite 服务器未重启
    修改配置后需重启开发服务器

  2. IDE 缓存未更新
    VS Code 可执行 > TypeScript: Restart TS Server

  3. 路径大小写敏感
    Linux 系统需确保实际路径大小写与引用完全一致

  1. 将高频访问目录设为别名（如 @components, @utils）
  2. 在项目文档中维护别名目录结构说明
