常见问题

Q1: 页面显示404？

  • 执行 hexo clean 清除缓存
  • 确认分类/标签页面类型设置正确

Q2: 图标不显示？

  • 检查主题配置的Font Awesome版本
  • 确保分类名称与配置完全一致

Q3: 部署失败？

  • 确认SSH密钥已正确配置
  • 检查仓库权限设置

提示：可使用 hexo server -p 5000 指定其他端口
